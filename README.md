# Tor Hosting Kit
### Tor (Browser) with only hosting

## Installing
### Go here and download:
### [Releases](https://github.com/Blue-Print-Company/Tor-Hosting-Kit/releases)
### Extract
### Start.bat to start
### Stop.exe to stop

## Building
### Connect to internet
### Download repo and execute Build.bat
### Open cmd and type:
### ```build [tor version]```
### Example:
### ```build 10.0.16```
### Or just start the batch file
