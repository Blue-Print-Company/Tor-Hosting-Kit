# Tor Hosting Kit
### Tor (Browser) with only hosting

## Installing
### Go here and download:
### [Latest Release](https://github.com/Blue-Print-Company/Tor-Hosting-Kit/releases/latest)
### Extract
### Start.bat to start
### Stop.exe to stop

## Building
### Connect to internet
### Way 1
#### Download repo and execute Build.bat
#### Open cmd and type:
##### ```build [tor version]```
#### Example:
##### ```build 10.0.16```
### Way 2
#### Just start the batch file
